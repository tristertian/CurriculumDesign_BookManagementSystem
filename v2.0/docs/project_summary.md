# 图书管理系统项目总结报告

## 项目概述

本项目是一个基于C++面向对象编程的图书管理系统，采用模块化设计，实现了完整的图书管理、销售处理、统计分析和数据持久化功能。

## 技术栈

- **编程语言**: C++11
- **开发模式**: 面向对象编程
- **构建系统**: Makefile
- **可选GUI**: FLTK
- **文档工具**: Markdown

## 核心功能实现

### 1. 图书管理模块 ✅

**已实现功能**:
- 添加图书：支持完整图书信息录入，ISBN号唯一性验证
- 查询图书：支持按ISBN、书名、作者、出版社四种查询方式
- 修改图书：支持任意字段修改，自动验证数据合法性
- 删除图书：按ISBN号删除，操作前确认

**技术亮点**:
- 使用智能指针管理对象生命周期
- 完善的异常处理机制
- 支持多种查询策略

### 2. 销售管理模块 ✅

**已实现功能**:
- 购买图书：输入ISBN和数量，自动计算总价
- 库存检查：购买前验证库存充足性
- 销售记录：自动生成销售记录，包含时间戳
- 库存更新：购买成功后自动减少库存

**技术亮点**:
- 原子性操作：库存检查和更新同步进行
- 销售统计：支持总销售额和销售记录统计
- 依赖注入：BookManager和SalesManager解耦

### 3. 统计分析模块 ✅

**已实现功能**:
- 显示所有图书：完整图书信息展示
- 按价格排序：从高到低排序展示
- 按库存排序：从多到少排序展示
- 按作者统计：分组显示同一作者的图书
- 按出版社统计：分组显示同一出版社的图书
- 综合报告：生成包含库存、价格、销售统计的完整报告

**技术亮点**:
- 使用lambda表达式进行高效排序
- 使用std::map进行分组统计
- 结构化的统计数据返回

### 4. 文件管理模块 ✅

**已实现功能**:
- 数据存盘：图书信息和销售记录分别保存
- 数据读取：支持从文件加载数据
- 自动备份：带时间戳的备份机制
- CSV导出：支持数据导出为CSV格式

**技术亮点**:
- 使用std::filesystem进行文件操作
- 异常安全的文件读写
- 支持自定义文件路径
